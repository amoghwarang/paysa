# Paysa

A simple client-side expense tracker web app.

## Overview

`Paysa` helps you track your daily expenses and savings goals. It is implemented as a lightweight static HTML app with no backend dependencies.

## Features

- Add and track expenses
- Set savings goals
- Inspect summary data in browser

## Usage

1. Open `index.html` in your browser.
2. Use the form to add transactions and goals.
3. View the generated charts and summary in the app.
