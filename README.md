# Paysa
https://amoghwarang.github.io/paysa/

This is a simple client-side expense tracker web app for tracking daily expenses and savings goals.

✅ Included Files
index.html — main tracker UI
expenses.html — expense details
goals.html — savings goals

📦 Features
- Add and track expenses
- Set savings goals
- View summary and charts

🚀 Setup
1. Open index.html in your browser.
2. Use the forms to add transactions and goals.
3. View summary and charts.

🧠 How It Works
- All data is stored in browser (no backend)
- Simple HTML/JS interface

🔐 Important
- No sensitive data is stored or sent

🛠️ Troubleshooting
- If data doesn’t save, check browser storage settings
- If UI doesn’t load, use a modern browser
